# Demo-Account-Instructions
How to setup a demo account with SafeStackAI

1. Create a new account with your Github Account here: https://app.safestackai.com/
2. Message you email account to the SafeStack team (Discord: @nickbrons.eth, Telegram: @nickbronsdoteth, Email: support@safestackai.com)
3. The SafeStackAI team will update your account shortly with demo repositories and credits - feel free to add your own repositories as well
